# Stationeers Mod Bootstrap

Automatic installer for Stationeers mod dependencies.

## Included

✔ BepInEx installer  
✔ StationeersLaunchPad installer  
✔ Auto Steam detection  
✔ Version update checking  

---

## Installation

1. Download:

    ```Stationeers_Mod_Installer_v1.0.zip```

2. Extract archive

3. Run:

    ```install.bat```

---

## Requirements

- Windows
- Steam installed
- Stationeers installed
- Internet connection
